# TRMNL Upcoming TV Releases

Monitor for new episodes of your favourite TV shows on your [TRMNL](https://usetrmnl.com).

## Running

Create `.trmnlp.yml` in the root of the project and add the `custom_fields` values as specified in `src/settings.yml`


Start the server:
```sh
trmnlp serve
```

Then visit http://127.0.0.1:4567/
